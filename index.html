Mockaroo API Data Visualization Tool

A simple web application to retrieve and visualize data through Mockaroo.com API

How to set up your Mockaroo.com schema in the visualization tool

1. Create and set up your schema in www.mockaroo.com/
2. Find your share link at the bottom of the schema page. Copy the last part of the share link and paste it in the schema link box.
3. Get your API key from your schema. API key is the characters after 'key=' attribute in the sample URL (curl).
4. Enter the key into API key text box and click the 'Test API' button. 5. Set the number of records to be retrieved from the schema (10 to 1000), pick X and Y variables and cilck 'Retrieve Data'.
